# 「笃行致远·拾光成纪」企划

> A Plan of Persistence and Memory

这是一个记录成长、分享回忆的综合企划网站，旨在通过多种形式记录我们的青春岁月，见证每一步的成长与蜕变。

![项目预览](logo.png)

## 项目简介

「笃行致远·拾光成纪」是一个精心设计的班级纪念网站，包含班级历史书、线上青春纪念册、定制同学录、入驻中国同学录以及音乐专辑等丰富内容。通过现代化的Web技术，将珍贵的青春记忆以数字化形式永久保存。

## 核心功能

### 主站
- **精美视觉设计**：采用渐变色主题和粒子背景效果
- **响应式布局**：完美适配桌面端和移动端设备
- **平滑导航**：固定导航栏支持平滑滚动定位
- **统计数据展示**：直观展示珍贵回忆、精彩活动等数据

### 青春纪念册
- **时间轴展示**：按照时间顺序清晰展现班级的成长历程
- **章节化内容**：分为六个章节，记录从七年级到毕业的完整历程
- **弹幕互动**：支持弹幕功能，增强互动体验
- **多媒体内容**：包含照片、视频等多种媒体格式

### 3D图片墙
- **沉浸式3D体验**：360度旋转的圆柱式图片墙
- **丰富交互功能**：支持拖拽旋转、滚轮缩放、点击查看大图
- **自适应布局**：五层圆柱结构，自动适配图片数量
- **大图浏览**：支持大图弹窗查看，支持缩放和平移

## 快速开始

### 环境要求

- 现代浏览器（Chrome、Firefox、Edge、Safari等）
- 本地Web服务器（推荐使用Live Server、http-server等）

### 安装步骤

1. **克隆或下载项目**
```bash
git clone [项目地址]
cd 重制版
```

2. **启动本地服务器**

使用Python：
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

使用Node.js（需要先安装http-server）：
```bash
npx http-server -p 8000
```

使用VS Code的Live Server插件：
- 右键点击index.html
- 选择"Open with Live Server"

3. **访问网站**

在浏览器中打开：`http://localhost:8000`

## 常见问题

### Q: 为什么需要本地服务器？
A: 由于浏览器的安全策略，某些功能（如图片加载）需要通过HTTP协议访问，直接打开HTML文件可能会受限。

### Q: 如何添加更多图片？
A: 将图片放入对应目录，脚本会自动扫描并加载。确保图片格式为jpg、png或jpeg。

### Q: 移动端体验如何？
A: 项目采用响应式设计，在手机、平板等移动设备上也能获得良好的浏览体验。

### Q: 如何修改统计数据？
A: 在 `index.html` 的 `stats-section` 部分修改对应的数字和标签。
